# SQL Retail Database Project

**SQL Retail Database Project** – Designed and implemented a transactional retail order management system using **SQL Server 2022**.  
Demonstrates skills in **database design, data integrity, T-SQL querying, analytical reporting, and performance optimization**.  
Includes **normalized schema, ERD modeling, business-focused queries, reporting views, and stored procedures**.  
A practical showcase of capabilities for **Junior SQL Developer, Database Support, and BI roles**.

---

## 📌 Project Overview
This project simulates a real-world retail order management system.  
It focuses on **data integrity, business reporting, and analytical insights** while demonstrating skills relevant to entry-level SQL roles in production environments.

---

## 🧱 Database Features
- Normalized relational database design  
- Entity Relationship Diagram (ERD) modeling  
- Referential integrity enforced with foreign keys  
- Constraints to enforce business rules  
- Sample data to simulate real-world transactions  

---

## 📊 Analytical Capabilities
- Multi-table JOIN queries  
- Aggregations, grouping, and subqueries  
- Common Table Expressions (CTEs)  
- Business-focused analytics including:  
  - Customer lifetime value  
  - Product and employee sales performance  
  - Identification of pending/problematic orders  

---

## 👀 Views & Reporting
- Financial summaries of orders  
- Sales performance dashboards (via views)  
- Customer and employee performance analytics
